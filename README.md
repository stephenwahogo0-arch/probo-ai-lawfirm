
# PROBO LAW FIRM: The AI Legal Singularity

Probo Law Firm is a production-grade, multi-agent legal reasoning system powered by the **VORTEX Core** and **IBM Quantum** integration.

## 🚀 Key Features
- **Quantum Law Engine**: Processes legal reality through Superposition and Entanglement to collapse probability spaces into a singular winning vector.
- **1,000 Real Agents**: Mobilized lawyer nodes and a specialized Trainer Agent managing the distributed legal mind.
- **Supabase Cloud**: Real-time cloud persistence for dossiers and agents with elite authentication.
- **Installable (PWA)**: Optimized for worldwide installation on iPhone, Android, and PC via QR code.
- **Botantical UI**: Professional "Eco-Natural" branding for an elite, immersive experience.

## 🛠 Tech Stack
- **Frontend**: React, Vite, TypeScript, Tailwind CSS v4, @supabase/supabase-js.
- **Backend**: Python, FastAPI, SQLAlchemy, Qiskit (IBM Quantum), OpenAI/Anthropic/Gemini SDKs.
- **Database**: Supabase (PostgreSQL).

## 📦 Setup & Installation

### 1. Database Initialization
Run the SQL found in `SUPABASE_SETUP.md` in your Supabase SQL Editor.

### 2. Backend (FastAPI)
```bash
cd backend
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
uvicorn app.main:app --reload
```

### 3. Frontend (Vite)
```bash
npm install
npm run dev
```
